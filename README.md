# Automatic TikTok Edit Maker for After Effects

This project is built to automate the TikTok video editing process using After Effects scripts to streamline video creation.

## Features
- Edit videos automatically
- Apply filters and transitions
- Manage audio selections

## Installation
1. Clone the repository.
2. Install required dependencies.
3. Run the main script to start editing process.

## Usage
- Use the command line to specify input video, filters, and settings.
